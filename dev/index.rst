########################
grommunio Developer Docs
########################

grommunio is a comprehensive communication and collaboration solution that
includes e-mail, calendaring, contacts, tasks, notes, video meetings, chat and
file management.

Audience
========

This document is for developers working with grommunio. It helps to integrate
and extend grommunio based on the developers own requirements.
